小米路由mini刷Padavan固件

小米路由mini支持家庭百兆光宽带和百兆以下带宽，百兆以上不支持。小米路由器mini的固件真心不好用，所以我刷完小米路由器3 http://huanglei.site/padavan ，就动手刷小米路由器mini，Padavan是目前最好的固件没有之一。固件在这里下载：http://opt.cn2qq.com/padavan Bootloader：https://breed.hackpascal.net

方法： 1、確保你的小米路由器已綁定小米賬號，下载mini可以用的Padavan固件到电脑：RT-AC54U-GPIO-30-xiaomimini-128M_3.4.3.9-099.trx 2、http://www.miwifi.com/miwifi_download.html 這里去下載 開發版ROM ，登录web后台，刷入下载好的开发版固件，去下面站点领取密码。 3、https://d.miwifi.com/rom/ssh ，領取你的ssh 密碼，用戶名: root，同时，下载对应提供的MIWIFI.bin的升级包到FAT16格式的U盘根目录上，插上U盘，断电，按住reset插电启动，等黄灯闪烁后放开，等重启之后，就可以SSH登录路由器了。获得SSH权限的方法也可以参考： http://huanglei.site/go/root 4、用ssh工具root登录路由器，刷入Uboot或者breed： cd /tmp && wget -O uboot.bin http://huanglei.site/uboot-xiaomi-mini-115200.bin 源路径：cd /tmp && wget -O uboot.bin http://downloads.openwrt.org.cn/PandoraBox/Xiaomi-Mini-R1CM/u-boot/uboot-xiaomi-mini-115200.bin mtd -r write uboot.bin Bootloader 或者 cd /tmp && wget https://breed.hackpascal.net/breed-mt7620-xiaomi-mini.bin mtd -r write breed-mt7620-xiaomi-mini.bin Bootloader 5、电脑用网线连路由器，网卡设置成192.168.1.5，子网掩码255.255.255.0，网关192.168.1.1 6、用别针或者牙签按住reset按钮并加电启动路由，等led灯橘黄快闪放开 7、电脑浏览器访问http://192.168.1.1，选择下载好的固件[RT-AC54U-GPIO-30-xiaomimini-128M\_3.4.3.9-099.trx](http://opt.cn2qq.com/padavan/RT-AC54U-GPIO-30-xiaomimini-128M_3.4.3.9-099.trx)上传并开始刷机 8、等待路由器重启就完成了刷机，将电脑网卡设置回自动 9、登录路由器访问http://192.168.123.1，用户名密码都是admin，设置路由器 这个固件无线性能卓越，cpu占用少，用来smb也很稳定，能开ssr，过滤广告，功能齐全。
