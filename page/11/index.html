牛蛙助手

发表于 2022-10-09 分类于 推荐

https://ios222.com

轻松签+只支持 A12-A15(xs - 13), 14.0 - 15.4.1 (15.5b4)；TrollStore也一样，像我这种iPhoneX（A11）停留在iOS13.7的用户只能Sideloadly一周一个人签。

牛蛙助手是一个在手机端实现个人签的工具，原则上可以一直对ipa文件实现每周一签，而且没有说挑设备和系统版本，很友好。

手机端下载安装地址：https://wwi.lanzoup.com/tp/i5Oit0h4rfuj

桌面端下载安装地址：https://wwr.lanzoui.com/ir8Pj0clfc6h

桌面端下载安装好之后用数据线连接手机，用桌面牛蛙安装手机牛蛙。只要手机上的牛蛙在，就可以一直签，如果误删，可以用桌面牛蛙重装一次，就像使用其他助手软件一样。两个方法同时存在可以避免其中一个方法失效，互为补充。牛蛙助手亲测靠谱，可以完美替代Sideloadly。

我正在使用的版本：https://ghproxy.com/https://github.com/oldj/SwitchHosts/releases/download/v4.1.2/SwitchHosts_mac_x64_4.1.2.6086.dmg

大疆无人机飞行模拟器

发表于 2022-10-02 分类于 爱好

2011年4月我入手了Parrot 的AR Drone 初代：https://m.weibo.cn/status/ey0mut 当时的大疆在深圳成立第5年，发布了Wookong-H悟空Helicopter飞控系统，还没有完整的无人机问世。直到2013年大疆发布第一款消费级无人机——大疆精灵Phantom 1号。时至2022年8月25日，大疆发布了DJI Avata穿越机。不得不说这个造型和11年前的AR Drone还是有几分相似。

作为起了个大早又赶了个晚集的初代无人机拥有者，我深知无人机操控是道坎。如果没有摸过模拟器就贸然实操，99%是首飞即炸机的结局。所以大疆给我们准备了模拟器，https://start.dji.com 桌面端打开可以用键盘操控，手机和平板端打开是虚拟摇杆，等键盘遥感熟悉了，可以换物理手柄去STEAM练FPV。

https://apps.apple.com/us/app/dji-virtual-flight/id1541992396
