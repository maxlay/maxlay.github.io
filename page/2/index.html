TeslaMate

发表于 2022-11-22 分类于 爱好

我的驾照是2002年获得的，我的第一辆车是2003年购买的本田飞度，第二辆车是2011年购买的别克凯越，第三辆车是2017年至今的奔驰GLA，第四辆车应该是几年之后的Tesla Model Q，或者Model Y等城市SUV车型。

Tesla Model Q在我看来定位更像是我最爱的本田飞度，飞度这辆车从2003年一直陪伴我到2017年，长达14年服役期间没有坏过，要提速有提速要省油又省油。相反，凯越这辆车是个油老虎，油门踩到发动机舱里都不带推背的，性价比完败飞度。所以凯越从2011年到2021年，只服役了10年，属于并不怎么及时得止损。GLA目前的感受属于介乎这两辆车之间，有加速有省油，按这两辆车在我人生中提供的上下限，GLA怎么也能陪伴10到17年，按13年算，2030年也许是我入手第一辆特斯拉电车的时间段，2030年的电池电控电机技术应该能比现在更成熟些，售价也能更合理些。

介于2030年我要购买特斯拉，所以提前做做技术储备，来看看特斯拉自持私有数据记录平台TeslaMate。

我理解抗衰老的形式有3种，逆转、冻龄、和延缓，今天在药研网公众号上看到了一篇文章，可能实现了当年秦始皇的愿望。现代医学就是科学，科学从来没有定论，科学就是在时间轴上，从后往前推翻的过程，所以请自行判断：

https://mp.weixin.qq.com/s/eTWgi9_suGQmngXSwdZ9UQ

该研究发现，PD-L1在衰老细胞中异质性表达，PD-L1阳性衰老细胞会随着年龄的增长而不断在体内积累。通过PD-1/PD-L1免疫检查点阻断疗法来清除PD-L1阳性的衰老细胞，或将成为一种颇具前景的抗衰老治疗策略。换句话说，治疗肿瘤的PD-1抑制类药物可能会像当年的可乐配方从咳嗽药水变成了碳酸饮料一样，从不能完全治愈肿瘤变成了可以逆转衰老的药物。PD-1抑制剂在对抗肿瘤细胞的过程中把衰老细胞当作了疾病给顺带手治了，听起来确实不可思议。其实药物这件事，从来都是有它的副作用的，就像网红司美格鲁肽的副作用是减重，FDA居然就批准了用它来减肥，哪一天FDA会批准哪一款PD-1抑制类药物用来逆转衰老，也不是完全不可能。

阿里云盘分享链接的集中管理

发表于 2022-11-17 分类于 推荐

阿里云盘不开会员，到期后因为没有剩余空间就无法保存别人分享的文件，虽然可以用插件来在线观看和直接下载别人分享的资源，但不能集中管理，我们用户看重的是阿里云盘里的资源，而不是阿里云盘给的空间容量虚数，没有资源的空间容量毫无意义。
关于不开会员集中管理分享资源，我的解决方案是用Mac桌面端Docker一个alist，用webdav挂载来实现分享文件的集中管理，你将拥有一个几乎无限容量大的只读资源库，可以随时全速查询、在线观看和下载资源。唯一的缺点是如果某资源失效，对应的文件夹就无法使用，所以还是建议对具有收藏价值的资源做好下载保存。但又有多少资源是具备收藏价值的呢，况且大多数分享链接既然分享出来就不会设置时效，可以一直使用。我把日常搜集到的阿里分享资源链接汇集到一个文件里并常驻在本博客页脚，名字是“AliyunPan”，将不定期维护。

我搜集维护的阿里分享链接集：

https://maxlay.github.io/aliyun.txt

Step1：

https://docs.docker.com/desktop/install/mac-install/
下载安装自己CPU对应版本的Docker Desktop on Mac

Step2：

https://alist.nn.ci/zh/guide/install/docker.html
安装发行版本

Step3：

查看管理员密码

Step4:

http://localhost:5244/@manage/storages
用管理员进入后台添加阿里云盘分享链接
需要填写刷新令牌、挂载路径和分享id这三个字段
刷新令牌获取说明在这里：
https://alist.nn.ci/zh/guide/drivers/aliyundrive.html
其中挂载路径和分享id一致能方便后期管理，一目了然。
添加AliyunPan是体力活，一个一个得把分享链接S后面的数组添加进去。
alist把存储配置在sqlite3数据库里，存储路径在”data/data.db”。
https://alist.nn.ci/zh/config/configuration.html#database
要是将来alist可以批量添加或者编辑一个配置文件那就太省事了。

Docker Desktop on Mac

发表于 2022-11-16 分类于 工作

在Mac桌面端管理家里的智能家电，我的方案是在MacOS上Docker一个Home Assistant。
一方面可以学习Mac下的Docker Desktop。
一方面可以再探索一下Home Assistant和Xiaomi MIoT插件。

Step1:

https://maxlay.github.io/2022/10/05/%E5%8A%A0%E9%80%9FGithub/

*确保可以访问Github（可选）

Step2：

https://docs.docker.com/desktop/install/mac-install/

下载安装自己CPU对应版本的Docker Desktop on Mac

Step3：

增加阿里的镜像源到“Preferences”里的“Docker Engine”

地址也可以自己去这里找：

https://cr.console.aliyun.com/cn-hangzhou/instances/mirrors

或者用Docker命令也能添加

创建一台安装有Docker环境的Linux虚拟机，指定机器名称为default，同时配置Docker加速器地址。

看机器的环境配置，并配置到本地，并通过Docker客户端访问Docker服务。

Step4：

在Docker阿里云镜像高速拉取并一键创建HomeAssistant到~/hassio目录

Step5：

打开http://localhost:8123/ 创建管理员并登陆

Step6：

从
https://github.com/ha0y/xiaomi_miot_raw/archive/refs/heads/master.zip
下载Xiaomi MIoT插件解压后
https://github.com/ha0y/xiaomi_miot_raw
按说明把custom_components目录复制到~/hassio下

Step7：

配置->设备与服务->添加集成->搜索Xiaomi MIoT->填写手机号和小米的密码->添加所有支持的设备
配置->设备与服务->添加集成->搜索HomeKit

Step8：

iOS和iPadOS端下载“家庭”APP，扫描HomeAssistant里HomeKIt二维码添加配件。

Docker核心技术原理以及应用:

https://www.aliyundrive.com/s/919XqEM8gGe

中国高血压临床实践指南

发表于 2022-11-13 分类于 健康

　　11月13日上午，《中国高血压临床实践指南》发布会于线上举行。该指南由以心血管科和高血压为主的多领域专家研究制定，形成了针对高血压的44个关键临床问题和102条推荐意见。 基于近年来大量高质量证据，针对临床较关心的高血压诊断界值、不同人群降压目标等内容，新版指南作出了重要调整。

130/80 mmHg！

高血压诊断界值下调

​ 新版指南推荐将我国成人高血压的诊断界值由收缩压≥140 mmHg和/或舒张压≥90 mmHg下调至收缩压≥130 mmHg和/或舒张压≥80 mmHg（1B）。

　　研究结果表明，我国40岁以下青年人群中，血压值为130至139/80至89mmHg的人群脑卒中风险，特别是脑出血风险明显增加。此次推荐高血压诊断标准改为130/80mmHg，对我国心脑血管疾病，特别是脑卒中的预防具有明确的必要性和可行性。
