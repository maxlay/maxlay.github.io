iPhoneX快充解决方案

发表于 2018-02-09 分类于 买货

知识点：

1、苹果所有自家的充电头中有5w（随iPhone附送）、12w（随iPad Pro附送）、29w（随MacBook附送）、61w、87w（随15寸MacBook Pro附送），其中仅后三种（使用usb-c接口）支持iPhone8/8+/X快充。 2、大于29w的充电头，与29w的充电速度相当，所以不用多花钱买更大功率充电头，请直接挑选29w的充电头。 3、购买快充套装需要：任何29w，并且支持PD协议的usb-c接口充电头搭配苹果自家usb-c 至 Lightning连接线，就可以实现半小时充满50%的快充功能。 4、使用12w的iPad充电头与29w的充电头，充电时间仅差15分钟。 5、使用转接头会使充电功率降回普通充电。 6、7.5w无线充电板的速度与5w充电头（随iPhone附送）速度一样，充满需要2.5个小时。

macOS High Sierra 允许任何来源

发表于 2017-03-02 分类于 爱好

在macOS Sierra 10.12系统以后，我们发现没有“显示允许任何来源”这个选项，解决方法是：

在终端输入：“

sudo spctl --master-disable

”

然后系统会提示输入系统密码，回车之后就可以安装受限制的”来自身份不明开发者”的程序。

本方法同样适用maxOS Mojave
