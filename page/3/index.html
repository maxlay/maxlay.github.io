从Manjaro到Fydeos

从DOS/Win32到Win95/98,从2000/2003到NT/XP，从win7到Win10，从MacOS到Debian/CentOS/Ubuntu，从Manjaro到ChromeOSflex/FydeOS。

2022年我的PC家用装的是FydeOS

当然，商用依然还是MacOS搭配iPadOS和iOS。

去年我玩过的Manjaro 新人可以参考：

https://mp.weixin.qq.com/s/hT5-bNDAlPlOkHIC1wnQ1A

Manjaro的安装笔记：

https://mp.weixin.qq.com/s/k0nHkq_ZY5YD-AyqYzyyGw

Manjaro功能强大，但对于一台老PC家用来说，仍然是过于庞大。

机缘巧合下，我看到了基于ChromeOSflex的国产项目FydeOS
FydeOS下载：

https://fydeos.com/download

FydeOS安装指南：

https://fydeos.com/docs/knowledge-base/getting-started/fydeos-for-pc

FydeOS（原名 Flint OS，于2018年6月正式更名）基于驱动 Chrome OS 及谷歌浏览器的开源项目二次开发，融合自主专利技术，继承 Chrome OS 所有特性，适配更多硬件品类，为中国学生而定制的本土化修改。

相对于传统操作系统，FydeOS有以下优势：

  • 极速响应永不变慢

  • 在一个 OS 平台上更好地运行网页程序+安卓程序+ Windows 程序+ Linux 程序

  • 账号数据云同步

  • 操作简单，安全稳定

  • 无打扰式更新

  • 在千元机上实现优质体验，流畅使用5-10年，超高性价比

FydeOS 是一款为大屏而生的操作系统，原生桌面操作系统，支持深度定制，PC、平板、物联网设备均可完美运行；我们为 FydeOS 设计了灵活的账号体系，可本地账号登录，也可对接其他云账号体系，实现数据云同步。

FydeOS可以在PC硬件上运行Android程序和Chromium浏览器，整个系统装完仅仅占用14.8G存储空间，如果你只是想用旧PC流畅的看看视频和浏览网页，那就不用犹豫了。

当然，它也能装Clash和SmartYouTube，下图是我的FydeOS的APP截图。

同时，我还在FydeOS启用了吃灰多年的遥控器，USB接收器插上就能用，不需要驱动程序。
操作PC方便的就像是操作一台安卓电视一样，而且性能格外强劲，PC变废为宝。

快捷指令一步到位

支付宝、微信、米家、QQ、淘宝、微博等等日常App都有自己的url scheme，可以用快捷指令和siri一步跳转到对应功能页面，实现方便的一键访问。

比如敲击手机后壳两下，触发对应的快捷指令，或者直接把快捷指令的图标放在桌面上，用siri触发，或者自动定时触发，十分便捷。

如何获得url scheme？可以点“添加到桌面”按钮，然后复制URL，一般是data:text/html;base64，开头，一串base64编码。

把base64之后的代码复制到Base64 在线编码解码 Base64 加密解密 - Base64.us（base64.us）解码后可以看到HTML，支付宝是alipay:// 开头，微信是weixin:// 开头，米家是mihome:// 开头，QQ是 mqq://开头，淘宝是taobao://开头，微博是sinaweibo://开头，把url scheme用快捷指令的safari浏览器打开，就能直接跳转，比如敲两下后壳打开带有疫苗完成的健康码，只需要一秒钟。

iOS常用URL Scheme附录 - 云+社区 - 腾讯云 (tencent.com)

有用有趣的URL Scheme，持续更新…… - 知乎 (zhihu.com)
