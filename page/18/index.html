电动自行车靠近解锁

发表于 2021-11-04 分类于 推荐

小2万的九号电动车E200有一个很牛的功能，就是手机靠近即可解锁，离开自动上锁。我觉得这个功能挺好，车也很好，就是售价不怎么好。

那么改造一下自己的电动车，获得这个功能需要花多少钱？ 答案是49.9元。 https://p.pinduoduo.com/WC2fOMKy 找到车子机械锁后面的红黄电线，割破线皮（不剪断），按图接线到芯片上，另外找一根常通的零线（黑色线）割破线皮（不剪断），同样按图接到芯片上，芯片会立马启动。 然后用安卓手机（不要连接名为Closer_switch的蓝牙，如果已连接就先取消配对）打开微信小程序“我的遥控器”，授权蓝牙权限，打开小程序后会自动连接芯片，连上后会自动转到设置界面。 修改参数，靠近值：5；远离值：8；模式：开关；旧密码：123456；新密码：改成你老婆的生日；点动时间：1。点“一键写入”。 写入成功后芯片会自动重启（如果没有自动重启，就重连一下红色火线帮她重启）然后用另一部手机（安卓和苹果都可以），通过蓝牙搜索Closer_switch，点连接，密码是你老婆生日。（第二第三第四部手机也是一样操作。） 连接过的手机靠近，车子的电源就开了，手机远离，电源就自动关闭。

一部手机连接过后，下一次靠近会自动连接。但是另一部曾经连过的手机就不会自动连接了，需要去蓝牙里点一下Closer_switch。同时两部以上有权限的手机靠近，它有几率会跳着连，需要非连接手机关蓝牙或者取消配对。另外，有些手机拨打电话时是使用4G通话，与蓝牙信号存在相互干扰，所以接打电话时，手机会偶尔断开与芯片的蓝牙连接导致锁车，所以需要接打电话就靠边停车。

快捷指令一步到位

发表于 2021-07-28 分类于 推荐

支付宝、微信、米家、QQ、淘宝、微博等等日常App都有自己的url scheme，可以用快捷指令和siri一步跳转到对应功能页面，实现方便的一键访问。

比如敲击手机后壳两下，触发对应的快捷指令，或者直接把快捷指令的图标放在桌面上，用siri触发，或者自动定时触发，十分便捷。

如何获得url scheme？可以点“添加到桌面”按钮，然后复制URL，一般是data:text/html;base64，开头，一串base64编码。

把base64之后的代码复制到Base64 在线编码解码 Base64 加密解密 - Base64.us（base64.us）解码后可以看到HTML，支付宝是alipay:// 开头，微信是weixin:// 开头，米家是mihome:// 开头，QQ是 mqq://开头，淘宝是taobao://开头，微博是sinaweibo://开头，把url scheme用快捷指令的safari浏览器打开，就能直接跳转，比如敲两下后壳打开带有疫苗完成的健康码，只需要一秒钟。

得了大病怎么治

发表于 2021-07-09 分类于 健康

1、无论选择中医还是西医治疗都必须先选择医生，最基础的也必须是正规医科大学毕业并遵守希波拉底誓言，有德有才的在院就业看诊的医生。当然，如果医生足够优秀，比如看肿瘤的医生，本身是csco的青年委员或者是副组长，那就更好了。（华山和中山的医生就是csco的人）
2、达到医保大病等级的疾病，有条件的，医保异地就医申报后，选择国内顶级医疗资源，北上广附近地区的群众可以就近选择对应大城市的专科医院。
3、比如我在浙江北部地区，距离上海90分钟车程，我就选择去挤兑上海的医疗资源，北京301或者302之类医院就显得鞭长莫及了。
4、每个医院都有各自的擅长，按科室的优秀程度选择对应的医院就诊，比如这次家人患病，我就先在本地三乙和三甲医院确诊，转院至上海复旦大学附属华山医院、中国人民解放军海军军医大学附属长海医院、复旦大学附属中山医院就诊，然后再转回本地私人医院（挂靠本地三甲医院，号称国际分院）来执行慢病治疗方案和尝试后期的康复治疗。前后经历六家医院。
5、最次的也都是三甲正规医院，对于那些都进不了医保的，先不怀疑其专业程度是否国家（医保）认可，甚至也先不看其以往病例口碑，连CT都不看就夸夸其谈，说在医院治疗不如归隐山林活的久的屁话，这么反科学反医学毫无人性的把病人的智商按在地上摩擦。为了骗几个救命钱毫无底线，希望这类骗子尽快归案，也希望能获得重刑（放古代，我希望这种货诛九族）。
