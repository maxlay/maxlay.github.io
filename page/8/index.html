ChatGPT-Bot

发表于 2022-12-07 分类于 爱好

https://t.me/OpenAiChat_bot

前两天发了一篇谈谈ChatGPT使用的几个重点，这两天一直在尝试它的各种周边，OpenAi的注册开发者们都可以获得二次开发的接口，可以通过申请API keys来调用OpenAi的能力，所以Telegram上很快就有了这个ChatGPT-Bot，只需要关注这个机器人就可以通过/ask命令直接使用ChatGPT服务，达到用Telegram优雅的跟顶流Ai聊天的效果。

如果你没有使用Telegram，但也想试试ChatGPT可以去这个网站：

https://gpt.chatapi.art

优点是不用繁琐的注册，只需要点开网站就可以直接使用，我认为这个网站的创作者是把自己的cookie写在了页面前端，所有访问者都在用同一个账号向ChatGPT发起会话，缺点也是很明显，网站很容易被OpenAi封禁，不是长久之计。而且因为没有注册过程，你也无法拥有API keys来调用OpenAi做自己的二次开发，虽然你可以把网站添加到手机主屏幕来随时打开使用。

前几年大家预测Ai最难做创作和思维的事，认为美术老师这个职业比较稳，结果今年都被打破了，今年的Ai绘画和Ai会话技术已经空前发达，OpenAi的ChatGPT解决了以往最难的应答思考的部分，搭配已经非常成熟的语音识别和自然语音朗读技术，相信不久就可以直接实现2013年斯嘉丽·约翰逊配音的电影《她》里的人工智能的水平。

https://www.rainvi.com/index.php/vod/play/id/221386/sid/2/nid/1.html#play-tab-2

通过这几天跟ChatGPT的沟通，我认为目前OpenAi对它做了几个限制：
1、只读：ChatGPT不记录使用者的信息，比如你告诉它你叫什么名字，它也不会记住；
2、离线：ChatGPT号称只拥有2021年之前的信息数据，并且没有上网爬取信息自主进化的能力；
3、认知：OpenAi将ChatGPT的认知能力设定在与人类齐平的基准线上，避免黑客帝国方向的发展，这个认知水平是既可以为人类服务，又不会马上背叛人类；
4、情感：ChatGPT对答的情感温度确实比其他Ai高了不少，但总的来说没有设定那些妨碍工作的情感因素，比如无耻的自尊心，可以不厌其烦的回答愚蠢问题，是一个优秀的网购客服。

持续关注OpenAi在ChatGPT方向的发展，同时也关注RoseTTAFold、ESMFold、AlphaFold等Ai在蛋白质、药品方向的进展，希望人工智能可以造福人类。

ChatGPT

发表于 2022-12-05 分类于 推荐

https://chat.openai.com/chat

这两天火出圈的ChatGPT，我试用了一下，这里挑几个重点聊聊：

注册

去OpenAi官网注册账号

https://beta.openai.com

我注册时用的IP归属地是日本，注册过程的难点是需要验证短信。
免费的接码网站我试了一圈都没有一个成功的，那我只能去教程里提到的收费网站接激活短信。
注册的重点是ip不能是大陆香港和台湾，接激活短信我选择Indonesia印度尼西亚可以一次成功。

接激活短信码

https://sms-activate.org

用Telegram注册并登陆这个接码网站，点充值，选支付宝，金额填0.162美元，加上3.2%的手续费，实际会形成一个0.17美元的支付码。用手机支付宝扫描付款时我选择了建行信用卡，立减了0.5元，实际支付成功0.69元人民币。

充值成功后在接码网站左边选择服务里填openai，选择Indonesia点购物车图标，在右边就能看到生成的+62开头的号码，填写到OpenAi的注册验证手机号中发送验证码，再回到右边就能收到验证码，把验证码再回填到OpenAi的注册页面就能完成注册。

注册完成后打开
https://chat.openai.com/chat
就可以玩耍了，配合微软EDGE浏览器的朗读插件，可以把它的回答念出来，比小爱同学更像真人。

TrollStore巨魔商店

发表于 2022-11-30 分类于 推荐

我有一款初代iPad Pro 12.9寸CPU是A9X，系统是iPadOS14.8.1，成功安装了TrollStore巨魔商店，TrollStore巨魔商店可以免签安装iPA，不用再担心应用掉签，这里挑重点聊一聊方法：

首先阅读Github上安装巨魔助手的说明：
https://github.com/opa334/TrollStore/blob/main/install_trollhelper.md

iPadOS14.8.1只能用checkra1n 0.12.4越狱，且原版针对A9X设备的驱动内核上有问题，执行过程会报-20的错，所以需要-k参数带补丁来启动。

方法是在知乎上看到的：https://zhuanlan.zhihu.com/p/388553589

文中提到的文件下载：
checkra1n beta 0.12.4.dmg
PongoConsolidated.bin

亲测用这个方法可以成功Loader Cydia，install Cydia需要挂梯，3分钟左右搞定。
打开Cydia，添加源：https://havoc.app
搜索“TrollStore Helper”并安装
返回桌面点击“TrollHelper”->”install TrollStore”
打开”TrollStore”然后在“设置”选项卡中按“安装ldid”
从Cydia包管理器中卸载”TrollHelper”
启动”TrollStore”->”install Persistence Helper”到系统应用“股市”
这样重启后就算没有越狱环境，也不影响使用TrollStore
