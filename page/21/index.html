家里连接光猫的主路由是小米路由3G，是一个千兆USB3的性价比产品，官网已经不卖了，要去淘宝或者闲鱼。这次依然用Padavan固件「http://huanglei.site/?s=padavan」来实现SMB服务，如果你不需要AdGuard Home ，我可能更推荐Hanwckf的固件「https://github.com/hanwckf/rt-n56u/releases」，比Padavan的固件「https://opt.cn2qq.com/padavan」更加小巧，用来SMB更稳定，网上也有Padavan和Hanwckf固件的网速对比。同样的硬件刷不同的固件，因为芯片的驱动程序和算法的不同，居然网速和稳定性会有很大差别，太玄幻了。另外，路由的USB口连接一个USB3扩展分线器，分线器上插了一个被格式化成EXT4的4G容量的2.0U盘用来opt，另外再插两个双盘位的USB3.0的硬盘盒，四块被格式化成EXFAT的2.0T容量的WD3.5寸硬盘用来存数据。这样做，防止频繁的opt调用机械硬盘，机械硬盘在不使用时可以休眠减少损耗。opt是用来开AdGuard Home等程序的，刷Padavan的魅力就在opt。

AdGuard Home 提升稳定性部署

发表于 2019-09-03 分类于 推荐

举个不十分恰当的例子，把互联网比作城市供水，每个地区自来水厂提供的水质和你所在管网末端位置的不同，有些家庭会明智的选择使用前置净水器，可以解决管道中的二次污染、软化水质、阻止水垢、防止锈蚀、保护净水器，当然还有太阳能和燃气热水器等等，但最终享受干净水的是人。精致的家庭还会使用厨下式厨余垃圾处理器，而落后的家庭甚至都不知道什么是前置净水器、什么是厨余处理器。所以总会有精致的家庭和落后的家庭，知识是永远不对称的存在着。互联网和供水一样，有大带宽和小带宽，有管网污染的问题，有被政府监控和被黑客偷窥的问题，还有烦人的广告问题，无论是视频前和视频中的广告，所以在入户光猫之后的主路由器上安装并使用私人DNS（AdGuard Home），就可以有助于“清洗流量”，虽然那些广告和监控程序不是DDOS的攻击，但一样是不可忽视的。

我在主路由的5353端口部署了AdGuard Home，另外，在家里的其他两个路由的53端口也部署了AdGuard Home，然后自定义配置文件 “dnsmasq.conf”

all-servers
dhcp-option=252,”\n”
server=10.1.1.2
server=10.1.1.3
server=127.0.0.1#5353 #AdGuardHome
no-resolv #AdGuardHome
dns-forward-max=1000 #AdGuardHome

按照这样的设置，路由器会同时查询内网的三台AdGuard Home，大大提高稳定性，三台垮一台甚至两台都不影响上网，而且私人DNS服务都在内网，ping值都小于1毫秒，独立硬件，解析速度也有保障，算是榨干所有路由硬件性能了。

私人DNS去广告AdGuard Home

发表于 2019-08-31 分类于 推荐

简单来说，你所访问的互联网，无论国内国外，无论有线无线，无论ipv4还是ipv6，在你发起访问时，都会最先在DNS上查询对应的服务器ip，然后网关再访问对应ip服务器上的内容，你访问一个网站，或者打开一个app，或者点播互联网电视，甚至喊一声小爱同学，在触发的刹那就需要至少一次访问DNS来查询服务器所在的ip，很多情况下你访问的内容又会被分散在多个服务器上，因为CDN的服务，每个URL都会有不同ip，每个ip可能又会对应多个域名，DNS会随着你的访问，频繁查询非常多次，一个家庭一分钟几万次DNS的查询很正常，所以DNS才是目前制约互联网访问速度的瓶颈。

小米盒子固件降级

发表于 2019-08-19 分类于 爱好

这两天做了一件不符合时间收益的事情，我把初代小米盒子和第二代小米盒子降刷到稳定流畅固件，这个操作的准备工作花去了很长的时间，目的也只是想变废为宝。
小米盒子第一代（MDZ-05-AA）和第二代（MDZ-06-AA）在2013年推出的时候，还是很引人注目的，当时官方售价300元，比时隔6年的4SE（MDZ-23-AA）的官方售价189元贵出不少，加上这几年的通胀和人民币贬值，该产品越来越廉价。
固件为什么要降刷，原因是小米盒子的UI越做越复杂，没用的功能越来越多，占用越来越大，导致越刷机升级固件越慢，甚至应用卡死，打不开，发热巨大等等，而且，对于我来说，没有Root的安卓设备是没有灵魂的。

我在想，也许小米就是靠把老产品的系统越做越卡，来促进新盒子的销售。

转换AppleMusic的DRM音乐

发表于 2019-08-13 分类于 爱好

之前有安利过苹果音乐的4个月试用，试用期间可以畅听苹果音乐的所有在线版权音乐，但是过期以后，需要续费每月10元，最少也要每年5元（学生价）

对于我来说，翻来覆去，也就听那么几首老歌，阿里系88会员有送“虾米音乐”年费会员，还有日常白嫖的“网易云音乐”和“QQ音乐”，几乎覆盖众新歌版权。

那么问题来了，怎么样乘免费畅听的4个月里，下载Apple Music里那些版权老歌，就是“虾米音乐”、“网易云音乐”和“QQ音乐”都没有版权的老歌？

解决方案：https://xclient.info/s/tuneskit-audio-converter.html

用TunesKit Audio Converter 可以一个一个都把iTunes里的DRM的AppleMusic内容转成MP3，机制大概就是软件用iTunes播放音乐，后台数字录制。

你可以把MP3上传到网易云音乐的60G“我的音乐云盘”。

你也可以上传到“百度网盘”，iOS端打开“百度网盘”首页点“音频”。

或者如果音乐数量少，直接放iCloud云盘也一样可以土法炼钢，又或者你本来就是6元每月的50GB/iCloud云用户也说不定。
