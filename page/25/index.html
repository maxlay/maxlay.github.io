我的2018双十一

发表于 2018-11-15 分类于 爱好

昨天收到今年双十一败的SSD硬盘，480G的容量，实测读写都在400MB/S左右。 用来替换2014年5月买的SSD硬盘，120G的容量，实测读写在250MB/S左右。 整整4年半的时间，现在主流SSD容量翻了4倍，同样规格的SSD因为3D闪存颗粒技术，读写速度也提高了不少，价格也便宜了。 之前的这块120G的SSD是用来替换MacMini的500G机械硬盘，自己动手拆机，买SSD硬盘连接线，给原本30-40MB/S的读写提高了10倍，原本的机械硬盘还是放在机器里做附属盘存数据用，容量也是500G左右。现在的机器总容量在1G左右，内存原厂4G，2014年也加到了顶配的16G。 硬盘换完之后，我是直接用TimeMachine做的恢复，时间大约用了20分钟，恢复完跟之前一模一样，就跟Ghost一样。苹果系统的设计理念之先进，确实不是巨硬可以觊觎的。 120G的SSD用来替换华硕PC在用的2011年9月买的64G金士顿固态，读写速度就不比较了。7年前64G的价格折算到每个G的话，今年的价格居然正好是当年的零头，当年是10.76元/G，今天是0.76元/G，相差14倍。真是『搜一颗赛艇』！

macOS Mojava 更换 壁纸

发表于 2018-10-31 分类于 爱好

macOS Mojava 更换难看的壁纸： 在桌面点击菜单上面的前往，选择前往文件夹，在路径这里输入路径： /Library/Desktop Pictures 点击前往后直达壁纸所在的文件夹，然后删除系统那些不好看的壁纸，保存入自己的壁纸，期间系统会要你输入登陆密码鉴定一下。 其中Mojave.heic就是锁屏壁纸，把自己最喜欢的图片名称改成Mojave.heic就可以变成锁屏壁纸，非常简单。 最重要的更换壁纸，还是在系统偏好设置的桌面与屏幕保护里选择就可以了。

我的壁纸在这个网站下载的：https://unsplash.com

YouTube视频下载方法

发表于 2018-10-18 分类于 推荐

下载油管视频只是为了当素材用，这里重申一下知识产权保护的重要性，不要下载版权视频用于商业目的，这里教的方法仅仅用于学术研究。 我的首选的工具https://www.clipconverter.cc 把视频的URL链接舔到文本框里，点Continue选择要下载的尺寸，支持4K视频，然后再点Start！等进度条跑完，就直接Download就可以了。

作为备选的工具https://www.converto.io 把视频的URL链接舔到文本框里，点Convert to:选择MP4，再点Convert等进度条跑完，点链接下载就可以了。

很少会用的工具https://www.findyoutube.net 这个工具主要是解析链接，不像是上面两个缓存、合成视频，当做备选吧。

M8SproL外贸电视盒

发表于 2018-10-05 分类于 买货

刚在某宝下单了油管和电报群里讨论很多的这款盒子MeCool的M8S pro L，3G运存32G存储，配置什么的看这里：https://post.smzdm.com/p/734369/ 这不是一款什么新产品，甚至油管上的评测已经是大半年前发布过的，选这款的原因是，原生安卓4K，不需要去折腾，不用亲自动手把手头的小米盒子3，就是MDZ-16-AA，刷成ATV，就能直接体验原生安卓，YouTube、Netflix。也有人拿他跟1499元售价的NVIDIA Shield TV比较：https://post.smzdm.com/p/635550/ 说起来，我今天把MDZ-05-AA的外壳拆了，用激光测温测了最热的地方，然后用硅脂和黄铜散热片重新加持，用透气耐高温粘胶包裹，直接放老电视上使用，老电视几乎不点播，就是用来看直播频道，插着百兆网线看电视家2.0没有任何问题，看了一下午，激光测温一直保持在45度左右。而原先那个乌龟壳是完全不能散热的，不能散热呢，CPU就会自动降频，最后死机。想不通当年小米的工业设计师是怎么想的，靠塑料壳传导散热太不科学了。 而这次M8SproL是用来替代原先的MDZ-16-AA的，而MDZ-16AA呢是去替换下MDZ-06-AA的。家里另一款MDZ-06-AA则由天猫魔盒的M16C替换，说起来天猫魔盒的配置总是比同期的小米盒子配置高。至此，家里退役出两台MDZ-06-AA，准备用来给老人看直播用，偶尔点播也没问题。当然，如果MDZ-06-AA可以刷固件成原生安卓就更好了，但是，太花时间了，我不准备虐自己。

盒子今天收到了，装了当贝市场，分数是小米电视3S的一倍，废话不说，上图。 另外，惊喜的是，M8SproL可以跟小米电视3S很好的配合解析和色彩，达到4K60hz，color444,8bit。 而且更好玩的是，我打开了盒子和电视的CEC功能，这样，用小米电视的遥控就可以控制盒子，盒子的遥控就可以下岗了，一个遥控搞定电视跟盒子。不知道什么是HDMI的CEC，可以自己百度，大概的意思就是借着CEC信号让用户可以控制HDMI接口上所连接的装置。如单键播放(One Touch Play)，系统待机(System Standby)。即是如果用户打开盒子，电视会由于CEC信号的通知而自动开机，然后视频通道亦会自动切换到播放器连接的通道上。而当用户关掉电视时，CEC信号亦会通知HDMI相连接的装置一同进入待机。由于这样，所以就可以完全变成单一遥控器控制所有HDMI连接的装置。

苹果MFi的USB-C/Lightning授权

发表于 2018-10-01 分类于 买货

苹果原有给第三方的USB-A to Lightning线的连接器C48将于2019年7月31日正式停止发货，取而代之的是C89。 苹果给第三方MFi授权的USB-C to Lightning线缆连接器是C94，是可以取得PPID（Product Plan ID），跟官方线一样可以18W的PD快充。 苹果149元官方第二版USB-C to Lightning线采用的连接器是C91。第一版USB-C to Lightning线采用的连接器是C52。 **牛逼的华强北确实没有能力破解32位动态码。经过大半年的市场检验，市场上没有一款第三方PD快充线可以在iOS11.3以后进行18W的PD快充，苹果放弃法律上了围剿破解线，直接在技术上设围栏，这招确实厉害。
