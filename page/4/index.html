TrollStore巨魔商店

发表于 2022-11-30 分类于 推荐

我有一款初代iPad Pro 12.9寸CPU是A9X，系统是iPadOS14.8.1，成功安装了TrollStore巨魔商店，TrollStore巨魔商店可以免签安装iPA，不用再担心应用掉签，这里挑重点聊一聊方法：

首先阅读Github上安装巨魔助手的说明：
https://github.com/opa334/TrollStore/blob/main/install_trollhelper.md

iPadOS14.8.1只能用checkra1n 0.12.4越狱，且原版针对A9X设备的驱动内核上有问题，执行过程会报-20的错，所以需要-k参数带补丁来启动。

方法是在知乎上看到的：https://zhuanlan.zhihu.com/p/388553589

文中提到的文件下载：
checkra1n beta 0.12.4.dmg
PongoConsolidated.bin

亲测用这个方法可以成功Loader Cydia，install Cydia需要挂梯，3分钟左右搞定。
打开Cydia，添加源：https://havoc.app
搜索“TrollStore Helper”并安装
返回桌面点击“TrollHelper”->”install TrollStore”
打开”TrollStore”然后在“设置”选项卡中按“安装ldid”
从Cydia包管理器中卸载”TrollHelper”
启动”TrollStore”->”install Persistence Helper”到系统应用“股市”
这样重启后就算没有越狱环境，也不影响使用TrollStore

HomeKit

发表于 2022-11-27 分类于 爱好

HomeKit是苹果2014年发布的智能家居平台，作为系统级智能设备操控，响应和操控便利性要优于应用级。在家里的软路由部署Docker版的HomeAssistant实现HomeKit需要避很多坑，这里浅谈一下几个重点。

硬件

首先我的软路由的硬件是2G内存版的R66S，TF卡存储，刷写固件比较方便。

日常用MacOS版的SD Clone对TF卡刷固件和备份固件。

固件

软路由的固件我选用flippy大佬20221018第8版启用6.0内核的固件，在店犀牛公布的百度云里有，名称是：openwrt_rk3568_r66s_R22.10.10_k6.0.2-flippy-78+.7z 分享链接: https://pan.baidu.com/s/1L3URkSfazKceYG2JarDE2w?pwd=6666

Docker

这个OpenWRT固件的Docker是20.10.17版，且不用调整分区扩容，上手就能用。
镜像加速器可以用：《Docker Desktop on Mac》里提到的配置，拉取homeassistant/home-assistant镜像大约10分钟可以搞定，然后新建容器的时候网络选HOSTS。

Homeassistant

启动容器里的Homeassistant，浏览器打开软路由ip的8123端口，因为用的是TF卡，读写都很慢，需要等，大约10分钟之后就会让你创建管理员身份并登陆。
然后像《Docker Desktop on Mac》里提到的那样下载解压Xiaomi MIoT，按说明把custom_components目录复制到Doker的/opt/docker/hassio目录下（这里我是直接sftp到软路由来实现的），随后再登陆Homeassistant里添加集成，搜索安装Xiaomi MIoT和搜索安装HomeKit。

Xiaomi MIoT

登陆配置Xiaomi MIoT里的小米米家账号可以自动同步“设备”，检查完整性，不完整可以多配置几次。

Homeassistant仪表盘

如果你想通过iPAD的Safari打开Homeassistant的仪表盘来日常控制设备也是可以的，也可以适当美化一下Homeassistant的仪表盘，自己按习惯的操作思路配置仪表盘，我既配置了仪表盘也启用了HomeKit。

HomeKit

用iOS或iPadOS里的家庭APP扫Homeassistant通知栏里的HASS Bridge：21064对应的二维码添加设备到HomeKit里，可以按设备属性新建房间来分类管理。也可以在家庭APP里添加自动化来创建设备的条件触发，怎么让自己觉得方便就怎么来。

也可以把iPad或者HomePod设置成中枢，手机脱离软路由网络环境也能继续控制家里的设备。

可以按自己的操作习惯把常用配件放在iOS或者iPadOS的电池下拉的“控制中心”里，实现系统级的管理设备。下拉点击，就像你平时调整音量和屏幕亮度那样方便。

TeslaMate

发表于 2022-11-22 分类于 爱好

我的驾照是2002年获得的，我的第一辆车是2003年购买的本田飞度，第二辆车是2011年购买的别克凯越，第三辆车是2017年至今的奔驰GLA，第四辆车应该是几年之后的Tesla Model Q，或者Model Y等城市SUV车型。

Tesla Model Q在我看来定位更像是我最爱的本田飞度，飞度这辆车从2003年一直陪伴我到2017年，长达14年服役期间没有坏过，要提速有提速要省油又省油。相反，凯越这辆车是个油老虎，油门踩到发动机舱里都不带推背的，性价比完败飞度。所以凯越从2011年到2021年，只服役了10年，属于并不怎么及时得止损。GLA目前的感受属于介乎这两辆车之间，有加速有省油，按这两辆车在我人生中提供的上下限，GLA怎么也能陪伴10到17年，按13年算，2030年也许是我入手第一辆特斯拉电车的时间段，2030年的电池电控电机技术应该能比现在更成熟些，售价也能更合理些。
