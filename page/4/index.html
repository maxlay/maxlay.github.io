Filza

发表于 2023-01-19 分类于 推荐

用iPad Pro刷抖音，发现抖音在iPad上不能横屏使用。抖音只在首次安装打开时可以横屏使用的，只要多刷几个视频关闭抖音第二次打开之后就无法横屏使用了，网上的教程都不好使。同时抖音会大量使用本地缓存，超过2.5G的缓存，不知道是不是在利用我的存储和带宽在做CDN。

那么怎么解决这个问题？

解决的思路简单来说就是用TrollStore来安装Filza，然后用Filza限制（App 管理器里找到的）“抖音”下面两个目录地址的写入和执行权限，限制权限前先删除这两个目录里的文档，不允许抖音缓存。问题就迎刃而解了。

Libary -> Application Support -> gurd_cache

和

Libary ->Caches

至此我的抖音占用的文稿和数据将永远停留在255MB左右。抖音视频会在内存里交换，不会保存到本地形成缓存，没有缓存就不会被当成CDN割韭菜。

要知道你手机存储颗粒的写入次数不是无限的，写入次数耗尽，硬盘就报废了，抖音把视频存在你手机里，根据算法会在你刷视频的同时把你手机里缓存的视频上传给你同网段的其他使用者，类似当年BT种子下载。这种技术可以大幅度减少抖音服务器的压力，把你的手机当成抖音的服务器。你的手机既要花费额外的电力，又要花费额外的上传带宽，这可都是你花的钱。

新冠感染

发表于 2022-12-28 分类于 健康

12月26日，国家卫健委发布公告，将新型冠状病毒肺炎更名为新型冠状病毒感染；经国务院批准，自2023年1月8日起，解除对新型冠状病毒感染采取的《中华人民共和国传染病防治法》规定的甲类传染病预防、控制措施；新型冠状病毒感染不再纳入《中华人民共和国国境卫生检疫法》规定的检疫传染病管理。对新型冠状病毒感染实施“乙类乙管”。依据传染病防治法，对新冠病毒感染者不再实行隔离措施，不再判定密切接触者；不再划定高低风险区；对新冠病毒感染者实施分级分类收治并适时调整医疗保障政策；检测策略调整为“愿检尽检”；调整疫情信息发布频次和内容。依据国境卫生检疫法，不再对入境人员和货物等采取检疫传染病管理措施。

新冠病毒真实影像

发表于 2022-12-20 分类于 健康

影像资料来源：https://ott.tsinghua.edu.cn/info/1002/1234.htm

新冠预防及治疗指南：

https://www.mojidoc.com/03dbc-fgimwhphhvabnbzbxigh6cdnji-00b/?utm_medium=sns&utm_source=telegram&utm_content=1219

2021-01-30 清华大学李赛团队实验室发布的新冠病毒真实影像，显示新冠病毒的平均直径大小是100纳米，也就是0.1微米，病毒需要通过附着在飞沫灰尘气溶胶等载体才能传播，搭载这些载体后的大小一般在5微米左右，可以理解成过滤PM2.5就能过滤PM5。

KN95口罩，主要是根据国家标准GB2626-2006《呼吸防护用品自吸过滤式防颗粒物呼吸器》的分类。口罩按照过滤元件的效率水平，分为KN和KP两类。KN适合防非油性颗粒物，非油性颗粒物如各类粉尘、烟和微生物等。KP适合防非油性和油性颗粒物，典型的油性颗粒物如油烟、油雾、沥青烟、柴油机尾气中含有的颗粒物和焦炉烟等。KN95口罩在用氯化钠颗粒物检测的情形下，其过滤效率应大于等于95%。而N95口罩是NIOSH(美国国家职业安全卫生研究所)认证的9种颗粒物防护口罩之一。“N”表示不耐油(not resistant to oil)。“95”表示暴露在规定浓度和粒径分布的专用试验粒子下，口罩材料的过滤效率应≥95%。

GB19083-2010《医用防护口罩技术要求》是医疗行业防护口罩技术标准的真正来源，通读整个国标原文，它和GB2626有很多类似的技术参数，比如，它也分了3个过滤效率等级，也检测非油性颗粒物过滤性（对比KN型口罩），但它直接舍弃了油性颗粒物的参数要求（即和KP型口罩完全不同），而1级为≥95%，最高级3级为≥99.97%，相当于直接对标KN95和KN100，没有KN90这个较低的级别。也就是说，符合GB19083过滤性能的最低标准，就相当于GB2626的KN95标准。我国GB19083里没有“N95”的说法，而是采用“1级”“2级”和“3级”的说法来表示过滤效率等级，一般1级就可以达到“N95/KN95”的要求。也就是说，只要符合GB19083标准的任何“医用防护口罩”，就绝对已经达到了N95级和KN95级的过滤效率了。

医用外科口罩标准YY 0469-2011《医用外科口罩技术要求》国标生产的口罩都是医用外科口罩，主要适用于有创操作使用，阻力低，阻血液、体液，细菌过滤效率≥95%。YY 0469-2011《医用外科口罩》YY/T 0969-2013《一次性使用医用口罩》标准分为GB国家标准，GB/T国家推荐标准，YY是医药行业标准的意思，YY/T是医药推荐标准。YY 0469-2011和YY/T 0969-2013两个标准有些类似，但是YY 0469-2011的要求和功能比YY/T 0969-2013要求更高。

口罩执行标准按严格程度排序依次是:

GB19083-2010 三级 = GB2626-2006 KN100 > GB19083-2010 二级 > GB19083-2010 一级 = GB2626-2006 KN95 > GB2626-2006 KN90 > YY 0469-2011 > YY/T 0969-2013

抗疫第四年的开局有感

发表于 2022-12-16 分类于 健康

湖州是浙江省下辖地级市，地处浙江省北部，东邻嘉兴，西依天目山，南接杭州，北濒太湖。湖州距离杭州大概80公里左右，距离嘉兴大概90公里左右。湖州处在太湖南岸，东苕溪与西苕溪汇合处，与无锡、苏州隔湖相望。高铁去南京一小时，驾车去上海两小时，放开疫情管控之后的人员流动不限制，疫情也就没有了限制。湖州的阳性比例虽然目前还不像上海北京那样，按现在新冠病毒的感染率，这也就是时间的问题，迟早的事，没有什么可侥幸的。

按照数学模型显示，中国14亿人口第一波将有60%人口感染，最终会有80%-90%的人口感染病毒，所以生活在这里想不受新冠病毒的影响是完全不可能的，要么感染一次让人体自身的免疫系统产生抗体，要么注射疫苗模拟一次感染来产生抗体，最终人类的基因都会被这次的病毒改写，这也是自然规律，不可抗力无法避免。而且病毒有200多个亚型变种，想感染一次不复感染也是不现实的，现在一周内反复阳性的报道也能看到。

作为普通老百姓的我，做两手准备来应对这次疫情，一方面接种中国解放军少将、工程院院士陈薇团队研发的吸入型加强疫苗，一周后产生抗体，一个月后达到最高的抗体水平；试试国产的这款疫苗牛不牛，就看我接下来羊不羊了。所以接种后的一个星期，我主动居家隔离，不给社会添乱，不挤兑第一波的医疗资源，把医疗资源让给危重病人。另一方面在家整理抗疫所需要的物资，做好日常防护，选好符合执行标准的口罩，医学规范消毒。

兵马未动粮草先行，有多少物资才能干多少事，巧妇难为无米之炊，所以今天整理家庭常备防疫物资，把没有标注符合执行标准的口罩放入后备储备箱，等这次疫情过去之后再日常使用，剩下合规的口罩按去什么地方，需要多少防范等级来分类使用。整理关键防疫药品和抗原检测试纸等。

与疫情战斗的这三年时间，数这一个月最危险，11月28日，美国驻中国的大使馆发公众号，要求在华美籍群众做好14天的医疗和隔离物资的准备。
https://mp.weixin.qq.com/s/as45CaW6m8hetCd3oIVPfQ
我也是在那时开始逐步囤药囤物资，那个时候的物流系统还能正常运作，接下来随着阳性比率的升高，物流企业的运力下降，物流只会越来越慢，甚至停滞，无法运作。但防疫物资的需求提升，反而更加剧了爆仓。本地药店没有药的话，网上也不会买的到，下单也不会有物流运力。

最后聊聊为什么第一时间接种吸入型疫苗，首先，我第一次接种疫苗是2021年5月，第二次是6月，第三次是12月，非常符合接种规律，但距离第三针剂一年之后的今天，抗体水平已经没有今年年中时那么高了，加上现在是爆发期，三年里最危险的三个月，所以急需提高抗体水平。

之前接种的三针都是北京生物的灭活疫苗，而这次作用呼吸道黏膜的给药方式，一方面是可以形成呼吸道粘膜防护屏障，一方面它的技术路线也是不同于灭活疫苗，吸入式的疫苗是用重组新型冠状病毒疫苗（5型腺病毒载体），经国家卫生健康委提出建议，国家药品监督管理局组织论证同意作为加强针纳入紧急使用。疫苗序贯接种过程中，由于人体接受了不同种类的疫苗，可能会使人体抵御新冠病毒的能力得到强化。因为通过接种作用机制不同的疫苗，能够不断完善人体抵御病毒的机制，从而有效降低人体受到病毒或疾病侵害的几率。
