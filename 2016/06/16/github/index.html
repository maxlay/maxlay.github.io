我的爱好

GitHub 神一样的存在

昨天把手头的iPhonePlus、iPad pro、MacBook pro更新到了iOS10和MacOS10.12，然而苹果抛弃了PPTP协议，而我之前在VPS上搭建的就是PPTP协议的VPN，好尴尬。今天上GitHub找了个支持OpenVZ的Cisco-IPSec-VPN-Install的shell，一分钟完成Cisco-IPSec-VPN的搭建。链接在这里：https://github.com/lokyshin/Cisco-IPSec-VPN-Install 这个脚本完美支持搬瓦工这样的OpenVZ技术的VPS。

搭建方法：“ wget –no-check-certificate https://raw.githubusercontent.com/lokyshin/Cisco-IPSec-VPN-Install/master/CIVInstall.sh && chmod +x CIVInstall.sh && bash CIVInstall.sh 2>&1 tee civaws_install.log ” 执行时会自动生成CA证书，会询问新建的账号名密码等。需要注意的是：

每次重启服务器后，不要忘了手动运行“bash startvpn.sh”，当然，也可以配置成自动启动。用户配置文件位置在/usr/local/etc/ipsec.secrets，可以自己添加账号，而每个账号也没有什么登录限制。
