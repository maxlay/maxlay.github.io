Windows11安卓子系统

发表于 2023-08-16 分类于 工作

https://maxlay.github.io/2023/08/12/%E7%BB%99%E5%8D%81%E5%B9%B4%E5%89%8D%E7%9A%84%E7%AC%94%E8%AE%B0%E6%9C%AC%E7%94%B5%E8%84%91%E5%8D%87%E7%BA%A7%E5%86%85%E5%AD%98/

上一篇文中提到《给十年前的笔记本电脑升级内存》总该有些原因，既然硬件妥了，上了 Win11 了，那就再多走一步，整点活，在 Windows 上安装安卓子系统，让安卓 APP可以装到 Win11 上来使用。

本着不重复造轮子的原则，优秀的教程推荐一下：

https://zhuanlan.zhihu.com/p/637883437

我没有按照教程的“Windows 11 WSA 安装 APK 方法”，而是在 Micrsoft Store 下载了“APK 安装程序”，用它直接安装 APP，软件会自动引导，只需要全程点击同意就可以了，简单高效。

延用教程中的总结：

与虚拟机或第三方安卓模拟器相比，微软官方的 Windows 11安卓子系统 在性能和与系统的集成上更为优越！非常实用，可以让安卓生态软件完美扩展到PC。
