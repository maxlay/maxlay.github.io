LanguageTool

Docker->LanguageTool

  • https://github.com/Erikvl87/docker-languagetool (Docker Hub)

or

  • https://github.com/silvio/docker-languagetool (Docker Hub)

如果 PULL 得比较慢，可以试着修改’Docker Engine’：

安装并且成功运行后，备份这个镜像可以执行
备份的镜像需要恢复可以执行

在 word 上使用

1. 下载安装Microsoft .NET Framework 3.5 Service Pack 1（完整软件包）

https://www.microsoft.com/zh-cn/download/details.aspx?id=25150

下载安装Microsoft .NET Framework 3.5 的中文语言包（可选）

https://www.microsoft.com/zh-cn/download/details.aspx?id=21891

下载安装Microsoft .NET Framework 3.5 （离线安装）

链接：https://pan.baidu.com/s/1SyEpLXguNyWrFMZuWFNaew
提取码：AQAQ

如果需要离线安装Microsoft .NET Framework 3.5，可以把下载好的 CAB 放在 D:\LanguageTool 目录里，然后在管理员身份打开的 CMD 里运行安装：

2. 下载安装Visual Studio 2010 Tools for Office Runtime

https://www.microsoft.com/zh-CN/download/details.aspx?id=48217

3. 下载安装languagetool-msword10-addin

https://github.com/jaumeortola/languagetool-msword10-addin/releases

它可以运行在Microsoft Word 2003, 2007, 2010, 2013, 2016 (Windows).

LanguageTool-msword10-addin本地服务的地址填
http://localhost:8010/v2

4. 用OUninstall.exe卸载完 2016 之前的版本（可选）

https://www.lanzouw.com/iDv5kw77egb

5. 下载 Office2016 专业增强版

https://www.123pan.com/s/ZrzA-kOZgh

6. 激活 Office2016 (HEU KMS Activator)

https://www.lanzouw.com/ibVCwuzvt2h

在 EDGE/Chrome 上使用 (无法脱机使用)

Grammar Checker & Paraphraser – LanguageTool

https://microsoftedge.microsoft.com/addons/detail/hfjadhjooeceemgojogkhlppanjkbobc

本地服务的地址填

http://localhost:8010/v2
